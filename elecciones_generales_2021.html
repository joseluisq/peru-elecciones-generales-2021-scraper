JSON Scraper de las Elecciones Generales Perú 2021

Un simple JSON Scraper de las Eleciones Generales Perú 2021 inpirado en NYT Vote Scraper.

Esta applicación web Hace un simple JSON scraping de la página de resultados de la ONPE sobre las Eleciones Generales Perú 2021 y crea un registro (commit) en un repositorio de Git.

El objetivo es poder ver el historial y las diferencias de resultados (diff) via el archivo elecciones_generales_2021.json.

El contenido de esta página es actualizada cada 30 minutos.
